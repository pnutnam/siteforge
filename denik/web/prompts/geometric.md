# Geometric Design Prompts

Use these prompts to generate geometric variations of the brand logo.

## Variations

### 1. Abstract Geometric
Transform the logo into abstract geometric shapes while maintaining brand recognition.
- Deconstruct logo into constituent shapes
- Use primary colors in bold geometric patterns
- Create repeating tile patterns
- Experiment with symmetry and asymmetry

### 2. Minimalist Lines
Strip the logo to essential lines and create minimalist compositions.
- Reduce to core linework
- Use negative space creatively
- Single-color or two-tone treatments
- Clean, modern aesthetic

### 3. Geometric Grid
Place the logo within geometric grid structures.
- Overlay grid patterns
- Create modular compositions
- Use geometric frames and borders
- Experiment with grid intersections

### 4. Organic Geometry
Blend organic curves with geometric precision.
- Flowing lines meeting sharp angles
- Gradient transitions between shapes
- Dynamic composition with balance
- Modern tech aesthetic

### 5. Bold Shapes
Amplify the logo with oversized geometric elements.
- Large background shapes
- Logo as focal point
- Dramatic scale contrasts
- Full-bleed color fields
